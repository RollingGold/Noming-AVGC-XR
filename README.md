# Noming-AVGC-XR

Third-person Action RPG prototype built in Unity 6.

## Overview

Noming-AVGC-XR is an action RPG prototype focused on responsive third-person combat, enemy AI, boss encounters, and scalable gameplay architecture. The project is designed around modular systems that can be expanded into a full RPG experience.

---

## Current Features

### Player

* Third-person character movement
* Camera-relative movement
* Sprinting
* Jumping
* Falling state detection
* CharacterController-based locomotion
* State machine architecture
* Combat animations
* Weapon attachment system
* Attack cooldown system
* Movement locking during attacks

### Combat

* Melee combat system
* Weapon hitbox activation through animations
* Enemy damage system
* Attack state handling
* Animation event integration
* Combat cooldown management

### Enemy AI

* NavMesh-based navigation
* Chase behavior
* Melee attacks
* Attack cooldowns
* State-driven animations
* Death handling

### Boss System

* Boss AI
* Multiple attack types

  * Normal Attack
  * Meteor Attack
* Multi-phase boss architecture
* Configurable phase health
* Optional damage carry-over between phases
* Boss death system

### Architecture

* PlayerMovement
* PlayerCombat
* PlayerStateMachine
* EnemyAI
* EnemyCombat
* EnemyStateMachine
* BossAI
* BossCombat
* BossStateMachine

Systems are built with modular components to support future expansion.
