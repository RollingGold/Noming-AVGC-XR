# Noming-AVGC-XR

Third-person action RPG prototype made in Unity 6.

## Features
- Third person movement
- Sprinting
- Jumping
- State machine architecture
- Combat system
- Animator integration
- Mixamo animations
- Weapon attachment system

## Tech
- Unity 6
- C#
- Unity Input System
- Character Controller
- Animator State Machine

## Planned Features
- Enemy AI
- Combo attacks
- Lock-on system
- Health system
- Damage system
- Bow combat
